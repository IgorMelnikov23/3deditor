ОПИСАНИЕ ПРОЕКТА:

Программа использует все известные возможности для 3D-моделирования разных программ (Blender, AutoCad, 3DsMax). В том числе, есть возможность создать анимацию, комментировать объекты на сцене, использовать дополнительные инструменты в виде плагинов, а также есть привязка к личному кабинету и возможность командной работы.

КОМПОНЕНТЫ:

1) Компонент "Сцена"

Компонент отображает все объекты проекта с учётом их координат, углов поворота, цветов и других свойств.

2) Компонент "Параметры"

Компонент отображает все параметры (свойства) объекта (цвета, координаты, углы поворота и другие) и позволяет настраить их.

3) Компонент "Дерево объектов"

Компонент отображает все объекты и зависимости между ними в древовидной форме. Данный компонент отображает ВООБЩЕ все объекты (даже те, что скрыты и не отображаются на сцене).

4) Компонент "Панель инструментов"

Компонент отображает все возможные инструменты для выбранного объекта и позволяет выполнить разные функции с помощью разных инструментов над выбранным объектом.

5) Компонент "Список плагинов"

Компонент позволяет выполнить поиск и использовать дополнительный инструмент для работы с программой, который не доступен в базовой комплектации.

6) Компонент "Поиск"

Компонент позволяет выполнить поиск объекта/объектов в проекте. В том числе, по регулярным выражениям.

7) Компонент "Ассеты"

Компонент позволяет находить и использовать в проекте уже готовые объекты с нужными параметрами (свойствами).

8) Компонент "Структура и настройки проекта"

Компонент позволяет просмотреть и изменить структуру всего проекта, в том числе, сцену. Можно настроить проект.

9) Компонент "История"

Компонент позволяет просматривать/сохранять/загружать проект в опрделённой точке разработки.

10) Компонент "Личный кабинет"

Компонент позволяет регистрироваться/выполнять вход/выполнять выход из личного кабинета. Наличие личного кабинета открывает дополнительные возможности редактора и позволяет получить доступ к проектам, которые расположены в облачном хранилище.

11) Компонент "Панель анимации"

Компонент позволяет создать/редактировать/просмотреть и удалить анимацию для любого объекта.

11) Компонент "Командная работа"

Компонент позволяет подключить/отключить других дизайнеров к проекту. Также есть возможность общения в реальном времени.

12) Компонент "Работа с переменными"

Компонет позволяет создать/удалить переменные, которые нужны для установки разных свойств (цвета, размеры, шрифты, угол поворота и другие).

ПРОГРАММНЫЕ СРЕДСТВА (инструменты):

- установка/редактирование/удаление разных объектов
- твёрдотельное моделирование для объектов
- группировка объектов
- перемещение по сцене в разных осях
- поиск по объектам (по названию)
- сохранение/чтение файла проекта
- регистрация/вход/выход из личного кабинета
- хранение файла проекта в облачном хранилище, в виде локальной копии
- возможность выбора объекта из ассета
- возможность выбора дополнительного инструмента из плагинов
- возможность сохранения объекта в виде ассета
- возможность добавить/просмотреть/редактировать/удалить анимацию для объектов
- возможность переключения 3D-сцены и 2D-сцены
- возможность изменить ось для просмотра объектов на сцене
- возможность настроить проект (название, автор, вид проекта)
- возможность командной работы над проектом, используя собственный протокол
- возможность создать/удалить переменные для разных свойств (цвета, шрифты)
- возможность включить/выключить сетку
- возможность выбрать привязки для установки и перемещения объектов
- возможность установить комментарий к объекту
- возможность использовать горячие клавиши с учётом разной клавиатуры для разных ОС
